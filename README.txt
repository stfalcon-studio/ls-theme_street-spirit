=== Установка и активация шаблона ===

1. В директорию ../templates/skin/ распаковываете содержимое архива ls-street-spirit-v0.x.x.zip
2. Если вы используете админ-панель aceAdminPanel, в конфигураторе выбираете шаблон,
   название которого совпадает с названием созданной Вами директории (в нашем случае — "street-spirit")

   Если используется стандартный конфигуратор, добавьте соответствующий параметр в файл config.local.php


   <?php
   // ../config/config.local.php

   ...

   $config['view']['skin']        = 'street-spirit';

   return $config;
